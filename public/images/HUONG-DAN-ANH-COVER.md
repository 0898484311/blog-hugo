# Hướng dẫn thêm ảnh Cover cho các bài post

## Danh sách ảnh cần thêm

Đặt các ảnh sau vào thư mục `static/images/`:

### Java Posts:
1. **collections.jpg** - Cho bài "Java Collections Framework"
2. **exception.jpg** - Cho bài "Exception Handling trong Java"
3. **multithreading.jpg** - Cho bài "Multithreading trong Java"
4. **oop.jpg** - Cho bài "OOP trong Java"

### JavaScript Posts:
5. **es6.jpg** - Cho bài "ES6+ trong JavaScript"
6. **dom.jpg** - Cho bài "DOM Manipulation trong JavaScript"
7. **async.jpg** - Cho bài "Async/Await trong JavaScript"

## Yêu cầu về ảnh

- **Kích thước đề xuất**: 1200x630px (tỷ lệ 1.91:1 cho social media)
- **Format**: JPG, PNG, hoặc WebP
- **Dung lượng**: Nên nén để tối ưu tốc độ tải trang
- **Nội dung**: Liên quan đến chủ đề của bài viết

## Cách thêm ảnh

1. Đặt ảnh vào thư mục `static/images/` với tên file đúng như trên
2. Ảnh sẽ tự động hiển thị làm cover image cho bài post
3. Cover image sẽ hiển thị ở:
   - Đầu bài viết (trang chi tiết)
   - Danh sách bài viết (thumbnail)

## Lưu ý

- Tên file phải khớp chính xác với tên trong front matter của bài post
- Nếu chưa có ảnh, blog vẫn hoạt động bình thường, chỉ không hiển thị cover image
- Có thể thay đổi tên file trong front matter nếu muốn dùng tên khác
